##  Built With

This project was built without heavy frameworks to demonstrate mastery of core web technologies and DOM manipulation.

* **Frontend:** HTML5, CSS3 (Custom Properties & Animations), JavaScript (ES6+)
* **Design:** Custom Dark/Cyberpunk Theme, Responsive Grid/Flexbox
* **Icons:** [FontAwesome](https://fontawesome.com/)
* **Fonts:** 'Outfit' and 'Space Mono' (Google Fonts)
* **Contact Service:** [EmailJS](https://www.emailjs.com/) (Serverless email handling)
